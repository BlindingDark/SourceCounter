# SourceCounter

统计 C/C++ 源代码的行数信息

## 环境依赖

依赖 `Erlang` `Elixir`，Windows 需前往官网下载安装，*nix 下建议使用对应的包管理器安装。

### ArchLinux

``` shell
sudo pacman -S erlang elixir
```

### MacOS

``` shell
brew install erlang elixir
```
